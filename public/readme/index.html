RicochetRobots

ボードゲームです．日本では「ハイパーロボット」と呼ばれています．基本的なルールは調べてください．

HTML5 Canvas + JavaScript + Firebaseで実装しています．最新のFirefox，Google chromeでの動作を確認しています．スマホ対応は未定です．最終的にはオンライン対戦を目標としています．ソルバーではないことに注意してください．

遊び方

左側のボタンをいい感じに操作して，白抜きの場所にロボットを帰してあげてください．

実装状況

  • （UIに文句を言わなければ）ローカルや画面共有で遊べるレベル

  • ゴールのランダム配置
